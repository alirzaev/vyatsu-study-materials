Эмулятор многоленточной машины Тьюринга
Указания к использованию
1. Для работы эмулятора требуется JRE (лучше - JDK). В операционных системах семейства Windows необходимо добавит в переменную окружения Path путь к папке bin JRE или JDK.
2. Все поля первоначального окна обязательны к заполнению.
3. Файл с расширением *.mtd должен содержать описание начального состояния лент (головка будет установлена напротив первого символа). Число строк должно совпадать с указанным числом лент.
4. Файл с расширением *.mtc должен содержать описание системы команд. Разделитель элементов - запятая (без пробелов). Символ ">" указывает на новые значения и сдвиги.
5. В каталоге examples содержатся примеры файлов *.mtd и *.mtc.
6. В настоящий момент эмулятор находится на этапе тестирования, поэтому высока вероятность ошибок в работе.  
7. Минимальное разрешение экрана - 800 пикселей по ширине (иначе возможны ошибки отображения).
